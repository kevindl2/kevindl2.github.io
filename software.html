Covid Contact Tracer

The above project was coded for the University of Illinois (Urbana-Champaign) class CS411 - Database Systems. The purpose of the project was to create a prototype covid contact tracing web application that used a MySQL database. In this application, users organize themselves into "bubbles" with other users, and record the date of each interaction with each bubble. If a user reports a positive covid test, all members of the bubbles that they are in are notified. In addition, a user can track their own vaccination status.

This app uses the Flask Python backend framework and the sqlalchemy Python library. It ran on a Google Cloud Platform instance.

Intro to Robotics - Final Project

The above project was coded for the University of Illinois (Urbana-Champaign) class ME 446 - Intro to Robotics. The purpose of this project was to demonstrate the ability to convert from joint space to a task space (placing a peg into a hole) , specify and follow a trajectory despite interference (moving through the zig-zag), and control a force vector applied in task space (placing the manipulator on top of the egg and pressing down).
